    API Quick Reference

    Core Endpoints

    • GET /health check backend availability.
    • POST /reset start a task episode with {"task_id": "easy"}.
    • POST /step submit one action payload.
    • GET /state fetch latest environment state.

    Tips

    • Use easy to validate payload format quickly.
    • Shift to medium/hard after policy shape stabilizes.
    • Clear logs when switching strategy to keep traces readable.
    • Track reward trend with session stats before hard resets.
